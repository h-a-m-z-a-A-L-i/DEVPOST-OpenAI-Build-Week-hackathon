# DEVPOST-OpenAI-Build-Week-hackathon

NotebookPilot is an autonomous Chrome assistant for local JupyterLab notebooks.
It detects the active notebook, reads normalized context, and performs notebook
operations through a JupyterLab frontend bridge.

## Project

- Chrome MV3 extension: `jupyter-chrome-agent/`
- Read-only notebook bridge: `jupyter-chrome-agent/bridge/`
- Gemini/Codex-compatible runtime: `jupyter-chrome-agent/runtime/`
- JupyterLab live-model bridge: `jupyter-chrome-agent/jupyterlab-bridge/`
- Team and architecture plan: `NOTEBOOKPILOT_PLAN.md`

Start with `jupyter-chrome-agent/README.md` for installation.
